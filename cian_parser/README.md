# CIAN PARSER

Change following fields in `settings.ini`:
- `url`
- `pages_with_flats`
- `links_to_check_path` (optional)
- `cian_houses_path` (optional)
- `user_agent`

To run cian-parser install `poetry` dependencies:
```bash 
poetry install
```
enter shell:
```bash
poetry shell
``` 
and run folowing command:
```bash
python main.py
```
